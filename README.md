# Тестовое задание

## Суть задания
Написать мок-сервис имитирующий ответ от https://www.cbr.ru/scripts/XML_daily.asp?date_req=02/03/2002

## Архитектура
mock_project1/  
├── Makefile         - Управление задачами (запуск, тесты, очистка)  
├── app.py           - Основной файл Flask-приложения  
├── mock_cbr.db      - SQLite-база данных (создается автоматически)  
├── requirements.txt - Зависимости проекта  
└── tests/  
....├── __init__.py   - Пустой файл для обозначения пакета  
....└── test_app.py   - Тесты с использованием pytest

## Описание решения
Это Flask-приложение имитирует эндпоинт ЦБ РФ:  
**`/scripts/XML_daily.asp`** — возвращает XML-ответ с текущими курсами валют, как это делает настоящий сервис.

- **Поддерживает** параметр `date_req` в формате `DD/MM/YYYY`
- **Возвращает** XML-документ с курсами 17 основных валют
- **Симулирует** ошибки: 20% запросов возвращают `500 Internal Server Error`
- **Кэширует** ответы в SQLite для отладки
- **Документирован** через **Swagger UI** — интерактивная документация в браузере
